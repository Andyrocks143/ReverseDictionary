Word Net Installation 

Download Wordnet from this link:

http://citylan.dl.sourceforge.net/project/wordnet30forwin/WordNet%203.0/3.0.2/WordNet_3.0_win32-setup.exe

and Install Word NET 

After installation, go to your installation directory(ex : C:\WordNet\3.0\dict) and change the following file names,

data.adj to adj.dat
data.adv to adv.dat
data.noun to noun.dat
data.verb to verb.dat
index.adj to adj.idx
index.adv to adv.idx
index.noun to noun.idx 
index.verb to verb.idx
